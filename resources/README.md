# Resources

Workbooks, templates, and printables that help you do the work of building a startup. Not Claude Code skills — these are documents you open, fill in, or print.

## Available resources

| Resource | What it is | Built by |
|---|---|---|
| [Discovery Interview Workbook](discovery-interview-workbook/) | Printable HTML companion for Rob Kenedi's 40-interview customer discovery challenge. Verbatim script, branching scenarios, pre-call briefing template, quick reference cards. | Sasha Madaniev (Protectyr), Cohort 7 |

## How to add a resource

Same idea as adding a skill — see [`../CONTRIBUTING.md`](../CONTRIBUTING.md). Resources live under `resources/<resource-name>/` with a README, the artifact files, and credit. Open a pull request or [start a Discussion](https://github.com/protectyr-labs/chaitech-ai-assistant/discussions) if you have something to share but are not sure where it fits.
