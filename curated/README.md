# Curated external skills

High-quality Claude Code skills authored outside this repo. We link rather than republish so authors keep control and credit. Install from the source.

## How curation works

A skill lands here when:

1. A cohort member or steward has used it in real work.
2. The original is maintained by the author (not abandoned).
3. The author permits linking and credits are clear.
